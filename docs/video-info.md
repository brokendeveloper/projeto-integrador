## Build demonstrável

O app pode ser demonstrado via **Expo Go**:

```bash
cd mobile
npm install
EXPO_PUBLIC_API_URL=http://SEU_IP_LOCAL:8000 npm run start
```

Para distribuição em APK, o próximo passo recomendado é gerar um build com EAS Build.
